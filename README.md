## 프로젝트 소개  
내 주식! 그때 샀다면 지금은 얼마일까?   
    
#### 🔗 [Back&Stock](https://backandstock.com/)
   
   
주식 백테스팅 서비스 Back & Stock 입니다.    
백테스팅 기간과 자산별 비율을 설정하면 수익률을 확인 할 수 있어요.   
한국의 코스닥, 코스피 상장 주식들을 백테스팅 할 수 있습니다.     
저장한 포트폴리오끼리 비교가 가능합니다.    
Back & Stock에서 백테스팅을 시작하세요!
